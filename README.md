# CRUD Task API

API REST para gerenciamento de tarefas (criar, listar, atualizar e deletar) desenvolvida com FastAPI e SQLAlchemy.

## Tecnologias

- FastAPI
- SQLAlchemy + SQLite
- Pydantic

## Como rodar o projeto

```bash
pip install -r requirements.txt
uvicorn main:app --reload
```

A API roda em `http://localhost:8000`. Acesse `/docs` para ver e testar todos os endpoints via Swagger.
